HTML

HTML, siglas en inglés de HyperText Markup Language, hace referencia al lenguaje de marcado para la elaboración de páginas web

HTML - Origen

La historia de lenguaje de marcado de hipertexto está estrictamente vinculada al científico inglés Tim Berners-Lee, quien es considerado el padre de la World Wide Web (W3) gracias a la investigación en el tema, que lideró desde 1980, mientras trabajaba para la Organización Europea para la Investigación Nuclear. El objetivo de Berners-Lee era crear un sistema hipertextual que permitiera conectar y compartir documentos entre diferentes computadores, sin importar el sistema operativo y aplicación web (navegador).
